===
Название: hostname
Описание:  программа, которая используется для установки или отображения текущего имени хоста, домена или узла системы. Эти имена используются многими сетевыми программами для идентификации компьютера. Имя домена также используется NIS/YP.
Синтаксис: hostname [имя программы] [опции]
Пример: hostname -i
Примечание: 

===
Название: nohup
Описание: позволяет  запускать команды, которые игнорируют сигналы
Синтаксис: /usr/bin/nohup команда [агрумент]...
           /usr/bin/nohup -p [-Fa] pid [pid]...
           /usr/bin/nohup -g [-Fa] gpid [gpid]...
           /usr/xpg4/bin/nohup команда [аргумет]...
Пример: nohup ./test.sh
Примечание: 

===

Описание:  Выводит первые строки файла
Синтаксис: head [опции] [файл] ...
Пример: head bot.py

===
Название: grep
Описание: печатает строки, подходящие шаблону
Синтаксис: grep [опции]... шаблоны [файл]...
           grep [опции]... -e шаблоны ... [файл]...
           grep [опции]... -f файл-шаблон ... [файл]...
Пример: grep -i 'net'  docker-compose.yml
Примечание: 

===
Название: chown
Описание:  Меняет у файла владельца и группу
Синтаксис: chown [опции]... [владелец][:[группа]] файл...
           chown [опции]... --reference=RFILE FILE...
Пример: chown tsire bot.py
